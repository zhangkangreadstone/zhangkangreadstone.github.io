Git@OSC的使用教程

  • 工具使用

Git@OSC的使用教程：

第一步： 使用准备
  1. 注册账号：http://git.oschina.net
  2. Git（可以在http://code.google.com/p/git-osx-installer/下载git安装程序，或者在https://www.kernel.org/pub/software/scm/git/下载源码安装
  3. 下载Xcode 并安装 略

  4. 创建工程目录 起个名字 例：Git@osChina

第二步：成生SSH密钥

打开终端命令工具，输入命令：ssh-keygen -t rsa -C "自己的邮箱@163.com"

终端输出：

在/Users/lsh/.ssh/id_rsa路径下这就生成了公钥和私钥

第三步：把myQQ.pub中的内容加入git@osc 的SSH密钥中

注意是带有.pub后缀的 用记事本打开

把里面的内容复制到下面的图框中

第四步：添加SSH并连接

输入命令：ssh-add 后面跟秘钥路径 是刚刚生成的密钥文件路径，屏幕输出：
Enter passphrase for 秘钥路径:输入设定的秘钥密码

输入命令ssh -T git@git.oschina.net，屏幕输出：
Welcome to Git@OSC , 自己名字!

第五步：设置个人信息

一旦你在自己的操作系统上安装完Git之后，你需要告诉Git你的用户名和电子邮件地址，从而创建自己的用户标识。你的所有代码提交都会与你的用户标识关联，这一点非常重要。

第六步：设置项目

输入命令：cd 项目文件夹

例：https://git.oschina.net/MyName/JCBTest.git
出处：

输入命令：git remote add origin git@git.oschina.net:MyName/JCBTest.git

输入命令：git pull -u origin master //从git@osc 上拉取项目

输入命令：git push -u origin master //提交项目

第七步：提交项目内容

cd gitOSChina/JCBTest/ 进入项目的目录
git init 初始化本地库（仅第一次时需要）
git remote add origin git@git.oschina.net:MyName/JCBTest.git 添加远程版本库的别名
git add . 添加所有文件到版本库
git commit -a -m “commit message” 添加所有修改该版本库
git pull origin master 从远程库获取并合并到本地库
git push origin master 将修改推送到远程库

第八步：更新工程

cd gitOSChina/JBCalendar/ 进入项目的目录
git add . 添加所有文件到版本库
git commit -a -m “commit JBCalendar” 添加所有修改该版本库
git pull origin master 从远程库获取并合并到本地库
git push origin master 将修改推送到远程库

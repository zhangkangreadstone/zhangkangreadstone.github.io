Git@OSC的使用教程

Git@OSC的使用教程：

第一步： 使用准备
  1. 注册账号：http://git.oschina.net
  2. Git（可以在http://code.google.com/p/git-osx-installer/下载git安装程序，或者在https://www.kernel.org/pub/software/scm/git/下载源码安装
  3. 下载Xcode 并安装 略
  4. 创建工程目录 起个名字 例：Git@osChina
  5. 配置Git并连接到git@git.oschina.net
第二步：成生SSH密钥

打开终端命令工具，输入命令：ssh-keygen -t rsa -C “自己的邮箱@163.com”
终端输出：

Enter file in which to save the key (/Users/lsh/.ssh/id_rsa): 这里填写公钥的名字(自定义)
Enter passphrase (empty for no passphrase): 输入自己设定的密码（不要忘记）
Enter same passphrase again: 确认密码
Your identification has been saved in myQQ.
Your public key has been saved in myQQ.pub.
The key fingerprint is:
SHA256:/gzPQdCnHkjs3qtU41dSSWnngWDBnsARgKwJU 113435533@qq.com
The key’s randomart image is:

在/Users/lsh/.ssh/id_rsa路径下这就生成了公钥和私钥

第三步：把myQQ.pub中的内容加入git@osc 的SSH密钥中

注意是带有.pub后缀的 用记事本打开

第四步：添加SSH并连接

输入命令：ssh-add 后面跟秘钥路径 是刚刚生成的密钥文件路径，屏幕输出：
Enter passphrase for 秘钥路径:输入设定的秘钥密码

输入命令ssh -T git@git.oschina.net，屏幕输出：
Welcome to Git@OSC , 自己名字!

第五步：设置个人信息

输入命令：

git config –global user.name “myName”

git config –global user.email myEmail@163.com

第六步：设置项目

输入命令：cd 项目文件夹

例：https://git.oschina.net/MyName/JCBTest.git
出处：

输入命令：git remote add origin git@git.oschina.net:MyName/JCBTest.git

输入命令：git pull -u origin master //从git@osc 上拉取项目

输入命令：git push -u origin master //提交项目

第七步：提交项目内容

cd gitOSChina/JCBTest/ 进入项目的目录
git init 初始化本地库（仅第一次时需要）
git remote add origin git@git.oschina.net:MyName/JCBTest.git 添加远程版本库的别名
git add . 添加所有文件到版本库
git commit -a -m “commit message” 添加所有修改该版本库
git pull origin master 从远程库获取并合并到本地库
git push origin master 将修改推送到远程库

ReactNative--NavigatorIOS

NavigatorIOS的使用简介

一个导航器对象包含如下的函数：

  • push(route) - 导航器跳转到一个新的路由。
  • pop() - 回到上一页。
  • popN(n) - 回到N页之前。当N=1的时候，效果和pop()一样。
  • replace(route) - 替换当前页的路由，并立即加载新路由的视图。
  • replacePrevious(route) - 替换上一页的路由/视图。
  • replacePreviousAndPop(route) - 替换上一页的路由/视图并且立刻切换回上一页。
  • resetTo(route) - 替换最顶级的路由并且回到它。
  • popToRoute(route) - 一直回到某个指定的路由。
  • popToTop() - 回到最顶层的路由。
